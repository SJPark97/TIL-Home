## Markdown

- Markdown (mark up 언어의 일종)
  
  - 텍스트 기반의 경량 마크업 (mark up : 역할 부여) 언어

- 제목 만드는 법
  
  - //# 제목 1
  - //## 제목 2
  - //### 제목 3
  - //#### 제목 4
  - //##### 제목 5

- 목록 만드는 법
  
  - //- 목록입니다 1
  - //- 목록입니다 2
  - //   -서브 목록 1
  - //   -서브 목록 2
  - //     -서브 서브 목록1

- 숫자로 목록 만드는 법
  
  - //1. 목록입니다 1
  - //2. 목록입니다 2
  - //    1. 서브 목록 1
  - //    2. 서브 목록 2
  - //       1. 서브 서브 목록 1

- 글씨 굵기 굵게 만드는 법
  
  - //**굵게**

- 글씨 기울이는법
  
  - //*기울임*

- 취소선 그리는 법
  
  - //~~취소선~~

- 인라인 코드
  
  - 'print("hello world")'

- 블록 코드
  
  - ```python
    for i in range(5):
        print(i)
    ```

- 수평선
  
  - //---

- 표
  
  - 마우스 우클릭으로 생성이 가장 쉬움
  
  - |     |     |     |
    | --- | --- | --- |
    |     |     |     |
    |     |     |     |
    |     |     |     |
